| OS | Windows Server 2022 Standard or Linux（CentOS, Rockyなど） |
| NIC | 1GbE×2 以上（10GbE追加も可） |

| 保守 | ProSupport 3年オンサイト保守 |

### 💰価格目安（本体＋構成）

- **本体＋構成**：80〜130万円（RAIDカード・SAS HDD・OS込み）
- **保守サポート付き**：＋10〜20万円
- **セットアップ費用別途**（10〜20万円程度）

## 📌補足：Dell導入時の注意点

- RAID設定やWindows ServerのCALライセンス別途考慮
